Speech Recognition Learning Framework for Non-Native English Accents (IEEE)

The AI-driven voice assistance systems have changed the way we interact with devices. Often when the user produces an accented command, these assistance systems aren't able to understand the user and thus produce an undesirable response which abates the user experience. We propose in this paper, a learning framework for speech recognition of Indian accent by analysing the features of Indian accented English and classifying based on sounds that are typical to Indian accented English.
