Projects

The best way to learn a technology is to implement it!

Quora Question Matching Problem

Designed and trained an ML model to match two questions with similar meanings. Task involved extracting various text features from the data corpus and solving the binary classification problem.

Netflix Prize

Worked on the Netflix’s personalized movie recommendation problem. The objective of the project was to predict the rating that a user would give to a movie that hasn’t been rated by the user.

Facebook Friend Recommendation System

Worked on Facebook’s graph dataset consisting of user network represented in a graph form. Objective was to recommend profiles that a user would like to follow based on their current network of followers and followees.

Stackoverflow Tag Prediction

Worked on Stackoverflow’s publicly available dataset which comprises of the question and associated tag. The objective was to predict tag(s) given the question. This is a multilabel classification problem as one question could have multiple classes (tags) associated to it.

Meetspeak: Auto-Transcription Tool

Meetspeak is a web application which allows the user to record a meeting and generate a quick transcript and summary of the meeting.
